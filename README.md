# Carpoo

### Go With The Flow

Carpoo is a modern ride-sharing and carpooling platform built to make travel more affordable, trusted and sustainable.

By connecting people already travelling on similar routes, Carpoo helps riders reduce travel costs while enabling drivers to share unused seats and recover fuel expenses.

---

## Why Carpoo?

Every day, millions of seats travel empty.

At the same time, students, professionals and daily commuters spend unnecessary money on transportation.

Carpoo bridges that gap by connecting riders and drivers already moving in the same direction.

The result is:

* Lower travel costs
* Reduced traffic congestion
* Better vehicle utilization
* More sustainable transportation

---

## Features

### 🚗 Rider Experience

* Search rides instantly
* Smart ride matching
* Ride alerts
* Driver trust score
* Secure in-app messaging
* Ratings and reviews
* AI assistance

### 🚘 Driver Experience

* Publish rides in minutes
* Vehicle verification
* Identity verification
* Passenger management
* Ride requests
* Driver trust score
* Earnings through seat sharing

### 🛡️ Safety & Trust

* Government ID verification
* Vehicle verification
* Driver & Rider Trust Score
* User reporting system
* Review system
* Secure communication

## Founder Note

Carpoo began as a student-built project driven by a simple observation:

Thousands of vehicles travel with empty seats every day while millions of people search for affordable transportation.

What started as an idea gradually evolved into a production application with real users, real feedback and continuous iteration.

Every feature, deployment, bug fix and release has contributed to the platform you see today.

This repository represents that journey.
